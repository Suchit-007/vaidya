# Design System: Vaidya.ai Clinical Platform
**Project ID:** `vaidya-clinical-rag-core`

## 1. Visual Theme & Atmosphere
The design system enforces a **Sanskrit Premium** aesthetic framework designed to elicit uncompromised domain credibility and rigorous analytical precision. The layout rhythm feels **Airy yet Formally Bounded**, pairing deep indigo-abyssal background substrates with high-refraction liquid glass surface containers. Interactive interfaces feel highly tactile and solid, intentionally avoiding flat corporate layouts by utilizing persistent visual anchoring, crisp typographic boundaries, and multi-textual source tracing to visually map clinical safety bounds.

## 2. Color Palette & Roles
* **Abyssal Slate-Black** (`#090D16`): Base viewport substrate layer. Absorbs excess visual noise to keep primary attention concentrated on extracted source snippets.
* **Sanskrit Ochre-Gold** (`#D97706`): Primary interactive state tone. Maps custom clickable keyword highlights, active button gradients, and accessibility outline boundaries.
* **Luminous Gold Glow** (`rgba(217, 119, 6, 0.35)`): Deep refractive support layer providing subtle, warm radial drop shadows underneath prominent components.
* **Aseptic Botanical Green** (`#10B981`): Assigned exclusively to verified zero-hallucination source pin borders, success tags, and high-confidence metrics.
* **Crimson Vitiation Red** (`#EF4444`): Functional attention tone utilized for low-confidence fallback warnings and real-time active voice listening feeds.
* **Pure Starlight White** (`#F1F5F9`): Clean primary contrast color deployed for standard text body streams.

## 3. Typography Rules
* **Header Architecture (`Cinzel`)**: Leverages sharp lapidary Roman/Sanskrit weighting (500 to 700 weight) to broadcast classical domain authority overhead. Character spacing is tightly bounded to maintain formal visual posture.
* **Systemic Body Text (`Outfit`)**: Highly readable modern geometric sans-serif optimized for intensive reading. Applied with ultra-light weighting (300 weight) for extensive clinical answers and medium weighting (500 weight) for embedded interface elements.

## 4. Component Stylings
* **Buttons:**
  - Shape: Smoothly bounded capsules featuring subtle interior radii (`rounded-xl` / 12px border radius).
  - Surface: Deep Sanskrit Ochre-Gold linear gradients backed by whisper-soft exterior illumination glow rings.
  - Tactility: Implements instant micro-scaling response curves (scaling down to 0.98 scale ratio on active user clicks) to guarantee intuitive mechanical confirmation.
* **Cards/Containers:**
  - Roundness: Subtly rounded corners with solid boundary definition (`rounded-2xl` / 16px outer radius).
  - Substrate: Semi-translucent deep slate panels (`rgba(15, 23, 42, 0.75)`) bounded by ultra-thin amber glass refractions (`border-glass`).
  - Depth: Lifted via multi-stage, high-contrast diffused drop shadows to separate clinical rationale from background layers.
* **Inputs/Forms:**
  - Stroke Style: Solid bottom focus border anchors augmented by absolute radial highlight rings on focus.
  - Surface: Smooth inner glass fills supporting high-contrast accessibility tags.
* **Custom Floating Overlay Tooltips (Interactive Hover Bubbles):**
  - Shape: Streamlined floating capsules with crisp micro-radii (8px corner roundness).
  - Background: Ultra-dense translucent glass matrices (`rgba(9, 13, 22, 0.95)`) bounded by precise gold accent strokes (1px width, `rgba(217, 119, 6, 0.25)`).
  - Depth: High-contrast whisper-soft diffused drop shadows (`0 8px 24px rgba(0,0,0,0.6)`) that lift the bubble above primary layers.
  - Micro-Motion: Programmed with an exponential ease translation curve (`translateY(-4px)`) coupled with smooth opacity transitions over a 150ms timing window.

## 5. Layout Principles
* **Whitespace Strategy**: Employs generous internal block padding (2rem section margins) to clearly separate runtime answers from verbatim source text extractions.
* **Symmetrical Center Alignment**: Core interaction panels float detached from external viewport frames, establishing absolute central focal dominance across responsive viewport metrics.
